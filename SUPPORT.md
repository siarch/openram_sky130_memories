# Support

## What This Repository Supports

This repository supports maintenance of SKY130 memory generation assets,
including configuration files, helper scripts, delivery packaging support, and
associated repository metadata.

## Good Requests

Use GitHub issues for:

- reproducible script failures
- packaging regressions
- documentation gaps
- CI failures
- workflow improvements

## Before Opening an Issue

Please include:

- the files or directories involved
- the command or workflow you ran
- the observed result
- the expected result
- any relevant logs or screenshots

## Not a Good Fit

This repository is not the right place for:

- private credential handling
- confidential infrastructure details
- upstream OpenRAM bugs that should be reported to the original upstream project
- unrelated toolchain support requests with no repository impact

## Preferred Paths

- Use the bug report template for failures and regressions
- Use the feature request template for repository improvements
- Use pull requests for concrete, reviewable changes

## Local Environment Notes

Machine-specific setup and recovery notes should stay out of public issues when
they expose local-only details. Keep private environment specifics separate from
general repository support requests.
